## Date and Time Format UI test suite

**Run for any question**

#### Test date and time format is changed in full log
- Answer a question
- Go to Settings and change date and time format
- Go to full log
    - Check the timestamp format is updated

#### Test date and time format is changed in question log
- Answer a question
- Go to Settings and change date and time format
- Go to question log
    - Check the timestamp format is updated

#### Test settings are saved after restart
- Go to Settings and change date and time format
- Restart the app
- Go to Settings
    - Check that date and time format are saved
